# Gemini

Open the cloned repo in Gemini CLI or a Gemini-based coding harness.

Start with:

```text
I am starting the OneMillion course. Read AGENTS.md, onemillion-builder/docs/teaching-protocol.md, and .gemini/GEMINI.md, then start Day 0. First verify fork, clone, origin, and upstream. Properly greet me, explain the course, provide copy-ready actions, and define what done means.
```

If your Gemini setup supports extensions, treat `onemillion-agents/` as the portable source of agent instructions.
