# Gemini

Open the cloned repo in Gemini CLI or a Gemini-based coding harness.

Run the installer first:

```bash
./course/install-agents.sh
```

Start with:

```text
I am starting the OneMillion course. Read AGENTS.md, course/course-manifest.json, course/docs/teaching-protocol.md, course/days/single.md, and .gemini/GEMINI.md, then start Day 0. First verify fork, clone, origin, and upstream. Properly greet me, explain the course, explain the OneMillion development pipeline, provide copy-ready actions, and define what done means.
```

If your Gemini setup supports extensions, treat `course/agents/` as the portable source of agent instructions.
