# Antigravity

Open the cloned repo in Antigravity.

Run the installer first:

```bash
./course/install-agents.sh
```

The installer creates:

```text
.agents/rules/onemillion-course.md
AGENTS.md
course/course-manifest.json
course/days/single.md
```

Start with:

```text
I am starting the OneMillion course. Read the workspace rules, AGENTS.md, course/course-manifest.json, course/docs/teaching-protocol.md, and course/days/single.md, then start Day 0. First verify fork, clone, origin, and upstream. Properly greet me, explain the course, explain the OneMillion development pipeline, provide copy-ready actions, and define what done means.
```
